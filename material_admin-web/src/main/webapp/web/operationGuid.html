邮件内容应包含:组织名称、负责人姓名、负责人学号、负责人所在学院

申请后两个工作日内会邮件回复您
用户注册:
在本系统中用户不是一个独立的个体,用户必须与组织相关联,在用户注册之前必须先申请对应的组织;
若您为组织负责人,请按照组织申请模块说明申请;
若您为非组织负责人,请您询问相应的组织负责人获取组织信息用于用户注册;
操作说明:
物资录入:在 组织物资 模块下
物资更新:在 组织物资 模块下
物资删除:在 组织物资 模块下
物资借出:在 组织物资 模块下
物资归还:在 我的借出 以及 组织借出 模块下都可进行归还确认
说明:物资借出以及归还都只能在物资拥有方进行操作,物资借入借出后应尽快在 我的借出 我的借入模块下核对是否准确
延迟说明:
组织借入以及组织借出会有五分钟的延迟
实时核对物资的借入借出情况应选择 我的借入 我的借出模块查看
模块功能说明:
物资查询:可根据物资名称(模糊匹配)、组织名称搜索可借物资情况
组织物资:所属组织的物资详细情况
我的借入:用户借入物资情况,含已归还记录
我的借出:用户借出物资情况,含已归还记录
组织借入:组织用户借入物资情况,含已归还记录
组织借出:组织用户借出物资情况,含已归还记录
我的:下面包含几个子模块:用户信息查询、用户信息更新、退出登录、用户注销
组员信息:所属组织所有组员的信息
